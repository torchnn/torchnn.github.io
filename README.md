# torchnn.github.io

torchnn 계정의 **루트 GitHub Pages** — 모든 앱이 공용으로 쓰는 도메인 루트 파일을 둔다.

- `app-ads.txt` — AdMob 퍼블리셔 인증(pub-8999074049545423). torchnn 전 앱 공용.
  크롤러는 마케팅 URL 도메인 루트(`torchnn.github.io/app-ads.txt`)에서만 찾으므로 여기에 둔다.
  새 광고 파트너(미디에이션) 추가 시 이 파일에 라인만 추가하면 전 앱에 적용된다.

정책·지원 페이지는 별도 레포 `torchnn/policies` (`torchnn.github.io/policies/`).
